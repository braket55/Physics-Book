Definition 1.1.1.
Scientific Notation
A standardized way to write very large or very small numbers using a coefficient between \(1\) and \(10\) multiplied by a power of ten. In scientific notation, a number has the form \(a \times 10^{n}\text{,}\) where \(1 \le |a| < 10\) and \(n\) is an integer. The exponent \(n\) indicates how many places the decimal point has been moved.
